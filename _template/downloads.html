Installation

1 Using a Package Manager

You can install ProDy and Evol using pip, pip install -U ProDy. Please note that you may have to manually remove previous versions (<1.10.3) of ProDy first.

2 Requirements
You need to install the following software before installing ProDy:
Python
We recommend that you use 2.7, 3.5 or later. For Windows, you can choose 32-bit or 64-bit Python installer. We recommend using Anaconda instead of the stardard Python distribution as it provides the conda package and environment manager as well as access to many useful packages including the interactive Python shell and Jupyter notebook.
NumPy
We recommend that you use the latest version of NumPy, but v1.10 and later are supported.
Biopython
We recommend that you use the latest version of Biopython, but all versions should be supported.
SciPy
We recommend that you use the latest version of SciPy, but all versions should be supported.
3 Installation
  • Linux & Mac OS
  • Windows

If you have pip, you can simply install ProDy as follows:

                    $ pip install prody

If you don't, you can manually download and extract tarball contents from PyPI website and run setup.py as follows: 

                    $ tar -xzf ProDy-{{ version }}.tar.gz
                    $ cd ProDy-{{ version }}
                    $ python setup.py build
                    $ python setup.py install
If you need root access for installation, try ``sudo python setup.py install``. If you donâ€™t have root access, see alternate installation schemes in Installing Python Modules.

Since version 1.10.3, ProDy does not provide a binary installer anymore. Instead, please install ProDy through pip:

                    $ pip install prody
4 Recommendations
We recommend that you also install Matplotlib for plotting, and IPython for interactive usage.

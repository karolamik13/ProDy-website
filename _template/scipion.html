Scipion ProDy plugin

Scipion-EM-ProDy enables access to main ProDy tools within the Scipion workflow engine for Cryo-EM. The latest version is available at https://github.com/scipion-em/scipion-em-prody and stable releases are in PyPI and the Scipion plugin manager.

Features

You can use Scipion-EM-ProDy to

  • parse PDB and mmCIF files from the PDB, select atoms, and perform ANM and RTB NMA
  • align a pair of structures and calculate deformation vectors
  • align a set of structures into an ensemble using most ProDy options and perform PCA to analyse energy landscapes
  • sample conformational transitions using ClustENM(D) with the option of fitting to cryo-EM maps

Reference
Krieger JM, Sorzano COS, Carazo JM Scipion-EM-ProDy: A Graphical Interface for the ProDy Python Package within the Scipion Workflow Engine Enabling Integration of Databases, Simulations and Cryo-Electron Microscopy Image Processing 2023 Int. J. Mol. Sci. 24:14245.
